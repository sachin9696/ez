# EZ Labs Assignment - React Application

## Overview

This project is a single-page React application replicating the Figma design for EZ Labs homepage.  
It includes multiple sections and a working, validated Contact Form connected to the backend API.

## Features

- Fully responsive UI for mobile, tablets, laptops, MacBooks
- Accurate Figma design recreation (Hero, About Team, About Us, Services, Portfolio, Contact)
- Contact form with front-end validation and error handling
- API integration with `https://vernanbackend.ezlab.in/api/contact-us/`
- Displays success message upon successful form submission
- Clean, modular codebase
